# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In summary: be respectful, be inclusive, be constructive.

## Reporting

If you experience or witness unacceptable behavior, please open an issue at [github.com/antoinebecker10-afk/trove/issues](https://github.com/antoinebecker10-afk/trove/issues) or contact the maintainer directly.

All reports will be reviewed and handled with discretion.
